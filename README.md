# Smart Study Session Tracker

This is a Python command-line program that helps users track study sessions by subject.

The program allows users to start and end study sessions, save study history, and calculate total and average study time.

## Features

- Start a study session
- End a study session
- Save the subject being studied
- Calculate study duration in minutes
- View study session history
- View total study time
- View average study time
- Prevent starting more than one session at the same time
- Exit the program

## How It Works

The program asks the user to choose from a menu.

When a study session starts, the program stores:
- the subject name
- the start time

When the session ends, the program:
- calculates how long the session lasted
- stores the subject in a list
- stores the session duration in a list

The user can then:
- view study history
- view total study time
- view average study time

## How to Run

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the terminal in the project folder.
4. Run the program using:
python
Smart-Study-Session-Tracker.py
5. Follow the menu options shown in the terminal.
